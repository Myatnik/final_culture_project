# Система управления отелем

## О проекте

Приложение для автоматизации работы стойки регистрации отеля.

Информационная система предоставляет администратору единую базу данных и наглядный интерфейс для отслеживания статусов жилых комнат в реальном времени, быстрой регистрации гостей и распределения задач между обслуживающим персоналом.

## Подготовка к запуску и системные требования

Перед установкой убедитесь, что на вашем компьютере установлены: 

**Python 3.8 или выше**

**Tkinter** (библиотека для графического интерфейса)

### Установка Tkinter

Если на устройстве нет Tkinter, его необходимо скачать для запуска программы

**Linux / WSL**

```bash

sudo apt update
sudo apt install python3-tk
```
