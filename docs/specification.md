# Спецификация

## Назначение системы

Система управления отелем для автоматизации работы ресепшена: регистрация гостей, управление номерами, учёт задач персонала.

## Функциональные требования

1. Регистрация гостя с автоматическим расчётом стоимости.
2. Управление номерами и их статусами.
3. Назначение задач уборки и ремонта персоналу.
4. Выселение гостя с обновлением статистики.
5. Поиск гостей по фамилии.

## Нефункциональные требования

 Python 3.8+

 Хранение данных в SQLite

 GUI через tkinter

 Разделение на ядро и интерфейс
