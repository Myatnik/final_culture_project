# Предметная область

## Сущности

- **Guest** — гость (имя, фамилия, контакты, история трат)
- **Room** — номер (номер комнаты, категория, этаж, статус)
- **RoomCategory** — категория (Standard, Comfort, Luxury, Economy) с базовой ценой
- **Stay** — проживание (гость + номер + даты + стоимость)
- **Staff** — персонал (уборщики, ремонтники, администраторы)
- **Service** — доп. услуга (завтрак, СПА, трансфер)
- **RoomTask** — задача для номера (уборка, ремонт)

## Статусы номеров
 `available` — свободен

 `occupied` — занят

 `cleaning` — требуется уборка

 `maintenance` — требуется ремонт

## Правила

1. Заселить можно только в `available` номер.
2. При заселении статус → `occupied`.
3. При выселении статус → `cleaning`.
4. Нельзя вручную менять статус занятого номера.
5. Стоимость = цена категории × ночи + доп. услуги × ночи.
